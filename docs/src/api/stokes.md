# Stokes / large-order diagnostics

Read the instanton action `S`, exponent `β`, prefactor `A`, and subleading
corrections `cⱼ` directly off the coefficient ratios `aₖ₊₁ / aₖ`, via
Richardson-accelerated tail extrapolation. See
[Stokes / large-order diagnostics](../tutorials/stokes_diagnostics.md) for
a worked example.
